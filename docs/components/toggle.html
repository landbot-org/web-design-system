Usage rules
• Use Toggle Switch for a single boolean setting (notifications on/off, dark mode, etc.). • Use Segmented Toggle when the user must choose between exactly 2–4 named options (Monthly/Yearly, Before/After, plan tiers). • Icons are optional. Only include them when they add meaning (✓ With / ✗ Without). Never use decorative-only icons. • The data-glider-color attribute on each button controls the glider color when that option is active. Values: success · error · brand · omit for default white. • The .seg-badge span is for contextual callouts like "Save 20%" — use only on the value-add option, never on both. • For pricing pages, use the default (white glider) or brand variant. The semantic error/success colors are reserved for comparison contexts.
