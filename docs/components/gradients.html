Usage Rules
✓ Do
  • Use brand gradient on CTA buttons and 1–3 bold (700) accent words in a headline
  • Use dark gradient for full-bleed dark sections — white text is always safe on it
  • Layer both glow gradients over the dark gradient for a rich hero
  • Use purple soft as a section background tint — standard text colors apply
  • Use gradient borders (2px gradient outline) to highlight a card
✗ Don't
  • Don't use brand gradient as a solid section background — contrast at the pink end is only 3.4:1 with white text
  • Don't use gradient text at regular weight (400) — thin strokes kill the effect; bold (700) only
  • Don't apply gradient text below 32px — too small to read the color shift
  • Don't use more than two gradient-heavy sections back to back
  • Don't invent new gradient combinations — use only the five tokens above
