⚠ What about Font Awesome?
Font Awesome still works and there's no need to remove it from pages that already use it. But don't add it to new pages — use Hugeicons instead. Over time, migrate away from Font Awesome as you rebuild sections.

How to add in Webflow

Three methods — choose the one that fits your situation.

✓ Recommended · Method 1
Webflow App (no code)
Install the official Hugeicons app directly from the Webflow App Marketplace. Icons appear as native Webflow elements — resizable, styleable, and bindable to CMS.
1. Open Webflow Designer 2. Apps panel → search "Hugeicons" 3. Click Install → done
Method 2 · CDN font
Link tag in <head>
Add the CDN stylesheet to Site Settings → Custom Code → Head. Then use class names in any Code Embed block.
<!-- Site Settings → Head --> <link rel="stylesheet" href="https://cdn.hugeicons.com /font/hgi-stroke-rounded.css"> <!-- Anywhere on page --> <i class="hgi-stroke-rounded hgi-home-01"></i>
Method 3 · Per icon
Inline SVG embed
Download the SVG from hugeicons.com and paste into a Code Embed block. Best when you need a single hero icon with specific colour or animation.
<!-- Webflow: Add element → Embed → paste SVG --> <svg width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor"> ...path data... </svg>

Which style to use

Hugeicons has 10 styles. For Landbot's marketing site, Stroke Rounded is the default — its rounded corners match the brand's rounded corners everywhere (buttons, cards, radius tokens).

Usage rules & do / don't
RuleWhy
Icons must always be a single colourMulticolour icons look inconsistent and increase visual complexity
Icon colour must match the surrounding text colourMismatched colours make the icon feel disconnected from its label
Never scale an icon below 14pxStroke icons become illegible under 14px — use text instead
Always pair an icon with a text label (or aria-label)Icon-only buttons are ambiguous and fail WCAG 1.1.1
Use Stroke Rounded as the default; Solid only for active/selected statesMixing weights randomly looks inconsistent
Keep stroke weight consistent within a sectionThinner icons next to thicker ones look like mistakes
Align icon to the optical centre of text, not the mathematical centreMathematical alignment often looks slightly low — adjust by 1–2px if needed
✓ Do
✓ Use Stroke Rounded for all standard icons ✓ Match icon colour to adjacent text ✓ Place icon in a tinted container for feature sections ✓ Use Solid style to show active/selected state ✓ Use the Figma plugin to insert icons while designing
✗ Don't
✗ Mix Stroke and Solid icons in the same row ✗ Use icons smaller than 14px ✗ Use Font Awesome on new pages ✗ Use multicolour icons in body content ✗ Use an icon-only button without an aria-label
