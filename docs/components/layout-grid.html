Content widths

Section backgrounds always go full-width. The content inside is constrained. Edge padding: 48px on desktop · 20px on mobile.

Standard
1200px — features, pricing, testimonials

Full bleed

Full-bleed sections still use a 1200px inner wrapper for text and buttons — the background bleeds, the content does not.
