Usage Rules
✓ Do
  • Use 1–2 dark sections per page — enough to create rhythm without feeling heavy
  • Always layer both glow gradients for depth — they're subtle but make a big difference
  • Use raised cards (surface-dark-raised) inside dark sections for feature callouts
  • Use gradient text on bold headlines — it pops especially well on dark
✗ Don't
  • Don't stack two dark sections back to back — always separate with a white or tinted section
  • Don't use --color-text-primary on dark — it's near-black and will disappear
  • Don't use the alt-purple tinted button (--color-surface: #EBECFF) on dark backgrounds
  • Don't use standard border color (#E5E6EA) — it disappears; use border-on-dark instead
