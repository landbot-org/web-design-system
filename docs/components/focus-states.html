Rules • Never use outline: none without providing a custom visible focus indicator. • Use :focus-visible, not :focus — this suppresses the ring for mouse users while keeping it for keyboard users. • Inputs use a border + shadow combo instead of outline, to avoid clipping inside containers. • On dark sections, switch to the white ring (--focus-ring-dark) — the purple ring has insufficient contrast on dark navy. • Make all interactive custom elements (cards, tags, toggles) focusable via tabindex="0".
