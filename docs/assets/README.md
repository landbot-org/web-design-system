# Brand assets

Local image files referenced by `docs/index.html` (referenced as `assets/<filename>`):

- `logo-primary.png` — primary brand logo
- `logo-outline.png` — outline variant
- `logo-dark.png`, `logo-dark-solid.png` — dark-surface variants
- `icon-dark.png`, `icon-outline.png`, `icon-white.png` — icon-only marks
- `wa-chat-background.png` — WhatsApp chat background pattern
- `g2-logo.svg` — G2 review badge

These travel with the design system so the showcase renders correctly. They're not part of the Claude skill payload — Claude reads tokens and component HTML, not images.
